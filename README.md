# ProyectoMD
Bienvenido a **ProyectoMD**, un proyecto final para el curso de "Introducción al Coding" dentro del programa "Mujer Digital". Este sitio web es una guía completa sobre los movimientos artísticos históricos, diseñada para ser tanto educativa como visualmente atractiva. Su objetivo es proporcionar a los usuarios una visión accesible de varios periodos del arte, sus características y obras destacadas.

## Descripción
**ProyectoMD** es una plataforma educativa basada en la web que explora la evolución del arte a través de sus movimientos más significativos. Construido como proyecto final para el curso "Introducción al Coding", sirve como una enciclopedia interactiva tanto para entusiastas del arte como para estudiantes. El sitio web presenta descripciones detalladas, líneas de tiempo, características clave y obras de arte destacadas para varios movimientos artísticos, comenzando desde el periodo barroco hasta estilos más contemporáneos.

Entre sus características principales se incluyen un sistema de navegación claro, diseños visualmente atractivos impulsados por Bootstrap y elementos interactivos para mejorar la experiencia del usuario. El proyecto demuestra una comprensión fundamental del desarrollo web front-end, utilizando HTML, CSS y JavaScript.

## Características
*   **Guía Completa de Movimientos Artísticos:** Información detallada sobre varios movimientos artísticos, incluyendo el Barroco, Rococó, Neoclasicismo y Romanticismo.
*   **Navegación Interactiva:** Un menú de navegación fácil de usar que permite un acceso sencillo a los diferentes movimientos artísticos.
*   **Galerías Visuales:** Cada sección de movimiento artístico presenta un carrusel que muestra obras de arte destacadas con sus descripciones.
*   **Diseño Responsivo:** Construido con Bootstrap, garantizando una experiencia consistente en diferentes dispositivos.
*   **Tipografía Estilizada:** Utiliza Google Fonts para elementos de texto distintivos y visualmente atractivos.
*   **Interruptor Interactivo:** Un botón mejorado con JavaScript para alternar la visibilidad del menú de navegación.
*   **Página Dedicada "Sobre Mí":** Proporciona información sobre los antecedentes y la motivación de la creadora.
*   **Información de Contacto:** Incluye enlaces a redes sociales y detalles de contacto de la creadora.

## Tecnologías Utilizadas
*   **Lenguajes:** HTML, CSS, JavaScript
*   **Frameworks/Librerías:** Bootstrap 5.3
*   **Fuentes:** Google Fonts (Josefin Slab, Sacramento, Tangerine)
*   **Otros:** Google Maps (para contexto de ubicación)

**Cómo usar:**
1.  **Página de inicio (`index.html`):** La página principal proporciona una visión general y enlaces a los primeros movimientos artísticos importantes. Puedes hacer clic en las imágenes o nombres para profundizar.
2.  **Páginas de Movimientos Artísticos (ej. `barroco.html`, `rococo.html`):** Cada página ofrece:
    *   Una descripción detallada del movimiento.
    *   Información sobre su lugar de origen o apogeo.
    *   Un carrusel de obras de arte destacadas con títulos, museos, técnicas y autores.
3.  **Menú de Navegación:** Haz clic en el logo en el encabezado para alternar el menú de navegación lateral, que lista todos los movimientos artísticos disponibles y un enlace a la página "Sobre Mí".
4.  **Página "Sobre Mí" (`sobre_mi.html`):** Conoce a la creadora, Areli Guevara, su pasión por el arte y sus estudios académicos.
5.  **Pie de Página:** Contiene información de contacto, detalles de derechos de autor y una cita motivacional.
